Test Logic Phân Quyền Hiển Thị Nút Xóa Comment

Logic Phân Quyền

Quy tắc hiển thị nút "Xóa":
1. Chủ bài viết: Có thể xóa tất cả comment trong bài viết
2. Người khác: Chỉ có thể xóa comment của chính mình

Test Case 1: Chủ bài viết xem bài viết của mình

Bài viết của User A (ID: user_a)
Nội dung bài viết...
Comment của User B (ID: user_b)
Nội dung comment... ✓ Hiển thị (vì User A là chủ bài viết)
Comment của User C (ID: user_c)
Nội dung comment... ✓ Hiển thị (vì User A là chủ bài viết)

Test Case 2: Người khác xem bài viết của User A

Bài viết của User A (ID: user_a)
Nội dung bài viết...
Comment của User B (ID: user_b)
Nội dung comment... ✗ Không hiển thị nút xóa (User B không phải chủ bài viết)
Comment của User B (ID: user_b) - User B đang xem
Nội dung comment... ✓ Hiển thị (vì User B là chủ comment)

Test Case 3: Replies

Bài viết của User A (ID: user_a)
Nội dung bài viết...
Comment của User B (ID: user_b)
Nội dung comment...
Reply của User C (ID: user_c)
Nội dung reply... ✓ Hiển thị (vì User A là chủ bài viết)
Reply của User C (ID: user_c) - User C đang xem
Nội dung reply... ✓ Hiển thị (vì User C là chủ reply)
